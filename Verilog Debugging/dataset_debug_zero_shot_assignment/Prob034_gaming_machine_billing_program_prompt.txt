- Boost mode: Activated when boost = 1, charges 2 yuan per clock cycle
- Balance deduction: Each clock cycle deducts 1 yuan (normal) or 2 yuan

- Warning indicators: Yellow light when remain > 0 and < 10, red light

- Continuous charging: Users can recharge during gameplay via set signal

Gaming machine operation:
- Normal mode: remain decreases by 1 each clock cycle
- Boost mode: remain decreases by 2 each clock cycle
- Charging: When set = 1, money value is added to remain
- Yellow warning: Active when 0 < remain < 10
- Red warning: Active when remain < 1 for normal mode, remaon < 2 for boost mode

- Reset behavior: Active low reset to initial state
