I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk
 - input  rst
 - input  data
 - output reg flag

The module should implement a Moore state machine using a two-stage
description method. The state machine has five states (S0, S1, S2, S3, S4)
and transitions based on the input data signal.

The state machine should work as follows:
- Input signal: data (wire type)
- Output signal: flag (reg type)
- Reset: rst is active low
- Clock: clk for synchronous operation

State transition table:
                   data
current state       1     0
  S0   next state  S1    S0
  S1               S2    S1
  S2               S3    S2
  S3               S4    S3
  S4               S1    S0

Output logic:
- When current state is S4, the output flag should be 1
- For all other states (S0, S1, S2, S3), the output flag should be 0
