I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk
 - input  rst_n
 - input  a (8 bits)
 - input  b (8 bits)
 - input  c (8 bits)
 - output d (8 bits)

In digital chip design, code that performs a specific function and is
relatively independent is usually written as a submodule. This submodule
is then instantiated in the main module when needed, to improve code
reusability and design hierarchy, and to facilitate subsequent modifications.

Design a hierarchical module structure where you first create a submodule
that compares two 8-bit unsigned numbers and outputs the smaller one.
Then, in the main module (TopModule), instantiate this submodule to find
the minimum among three 8-bit unsigned inputs.

The submodule should take two 8-bit inputs (data_a and data_b) and output
the smaller of the two values. In the main module, use this submodule to
determine the minimum value among a, b, and c.

The rst_n signal is an active-low asynchronous reset signal. Assume all
sequential logic is triggered on the positive edge of the clock.
