
I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  ALUOp   (2 bits)
 - input  Funct7  (7 bits)
 - input  Funct3  (3 bits)
 - output Operation (4 bits)

The module should implement an RISC-V ALU control decoder. It is purely
combinational and generates a 4-bit Operation code from ALUOp, Funct3,
and Funct7 to select the arithmetic/logic/shift/compare behavior in the ALU.
