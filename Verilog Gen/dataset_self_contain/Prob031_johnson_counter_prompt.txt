I would like you to implement a module named TopModule with the following
interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk
 - input  rst_n
 - output reg [3:0] Q

The module should implement a 4-bit Johnson counter (also known as a twisted
ring counter). Assume all sequential logic is triggered on the positive edge
of the clock. The reset input is active low asynchronous, and should reset
the counter to 0000.

The Johnson counter should work as follows:
- The counter follows a specific sequence of states in a circular pattern
- The state sequence is: 0000 -> 1000 -> 1100 -> 1110 -> 1111 -> 0111 -> 0011 -> 0001 -> 0000
- This creates a cycle of 8 states before repeating
- On each positive clock edge, the counter advances to the next state in the sequence
- When rst_n is low, the counter should reset to 0000
