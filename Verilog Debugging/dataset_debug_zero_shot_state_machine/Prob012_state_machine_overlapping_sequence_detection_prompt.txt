interface. All input and output ports are one bit unless otherwise
specified.

 - input  clk
 - input  rst
 - input  data
 - output reg flag

The module should implement a state machine that detects the sequence "1011"
with overlapping detection capability. The state machine should detect
multiple occurrences of the sequence even when they overlap. reset is active low.

The sequence detection should work as follows:
- Monitor the input data stream continuously
- Detect when the sequence "1011" appears in the input
- Support overlapping detection: "10110111" should be detected twice
- Output flag signal should be high for one clock cycle when the sequence
  is detected, starting from the next clock cycle after detection
